Our capabilities

Categories for the business services HMCTS provides across the justice system.

Each API published on the Marketplace sits within one of the following capabilities. Use these categories, and the domain filter in the API Catalogue, to find the right API for your integration.

Case Administration

Creating, updating and retrieving the administrative record of a case — status, parties, and case type — across crime, civil, family and tribunal jurisdictions.

View related APIs ›

Hearing Results

Recording and retrieving the outcome of a hearing, including orders made and results recorded once a hearing has concluded.

View related APIs ›

Scheduling and Listing

Publishing and retrieving court and tribunal listings — which cases are being heard, where, and when.

View related APIs ›

Reference Data

Canonical, shared reference data used across HMCTS services — courts, venues, case types and jurisdiction codes — so teams don't each maintain their own copy.

View related APIs ›

Financial

Fee calculation and payment transaction data for court and tribunal fees.

View related APIs ›
