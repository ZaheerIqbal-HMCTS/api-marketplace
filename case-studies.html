API platform case studies

Examples of how teams have integrated their software with HMCTS APIs.

Reducing manual lookups in a case progression tool

A regional court administration team integrated the Crime Prosecution Case Details API into their internal case tracking tool, replacing a manual daily export process. Staff can now look up live case status and hearing history directly, cutting a routine daily task from around 40 minutes to under a minute.

API used: Crime Prosecution Case Details API · Environment: Production

Publishing daily listings to a public-facing service

A digital service supporting self-represented litigants used the Court Listings and Scheduling API to show upcoming hearing dates without needing court staff to re-key published cause lists. The integration went from sandbox testing to production access in under three weeks.

API used: Court Listings and Scheduling API · Environment: Sandbox → Production

Automating fee calculation in a case management system

A tribunal case management team used the Fees and Financial Transactions API to calculate the correct fee for a case type automatically at the point of submission, reducing fee-related rework and manual reconciliation later in the process.

API used: Fees and Financial Transactions API · Environment: Production

These are illustrative examples. If your team has integrated with an HMCTS API and would like to share your experience, contact the marketplace team.
